# G-Express 서비스란?

G-Express 서비스는 고객이 운영하는 인터넷 어플리케이션 서비스의 글로벌 이용자에게 전용망 기반 고품질 가속서비스를 제공하는 글로벌 가속 서비스 입니다. 이를 통해 인터넷 어플리케이션 서비스의 패킷 손실과 서비스 전송 지연을 최소화 하고 사용자 경험을 향상 시킵니다.
